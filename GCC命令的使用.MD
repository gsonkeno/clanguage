## gcc --help

|  参数   | 说明  |
|  ----  | ----  |
| -S  | Only run preprocess and compilation steps |
| -o &#60;file&#62; | Write output to &#60;file&#62; |
| -c | Only run preprocess, compile, and assemble steps |
| -E |  预处理 Only run the preprocessor |
| -O2 |  编译2级优化  |
| -g  |  生成调试信息 |
